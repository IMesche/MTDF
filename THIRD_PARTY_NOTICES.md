## Software

### CLASS (Cosmic Linear Anisotropy Solving System)

The `class_mtdf/` directory contains a modified version of CLASS v3.2.

- **Authors:** Julien Lesgourgues, Thomas Tram, and collaborators
- **License:** GPL-3.0
- **Repository:** https://github.com/lesgourg/class_public
- **Citation:** Blas, D., Lesgourgues, J., Tram, T., "The Cosmic Linear Anisotropy Solving System (CLASS). Part II: Approximation schemes", JCAP 07 (2011) 034, arXiv:1104.2933
- **Modifications:** Documented in `class_mtdf/MODIFICATIONS.md`. Two new source files (`mtdf.c`, `mtdf.h`), modifications to `background.c`, `input.c`, `perturbations.c`, `background.h`, and `Makefile`.

### HyRec-2

Bundled within CLASS as `class_mtdf/external/HyRec2020/`.

- **Authors:** Yacine Ali-Haimoud and Chris Hirata
- **Citation:** Ali-Haimoud, Y. and Hirata, C.M., "HyRec: A fast and highly accurate primordial hydrogen and helium recombination code", Phys. Rev. D 83, 043513 (2011), arXiv:1011.3758

### Cobaya

Used for MCMC sampling in Phase 5 (not bundled; installed via `scripts/install_cobaya.sh`).

- **Authors:** Jesus Torrado and Antony Lewis
- **License:** LGPL-3.0
- **Citation:** Torrado, J. and Lewis, A., "Cobaya: code for Bayesian analysis of hierarchical physical models", JCAP 05 (2021) 057, arXiv:2005.05290

### CosmoPower (emulator cache files redistributed via Zenodo)

Used to replace direct CLASS Boltzmann-solver calls during Phase 2 MCMC sampling. The CosmoPower library itself is **not bundled**; only generated emulator cache files are shipped as a separate Zenodo deposit.

- **Authors:** Alessio Spurio Mancini, Davide Piras, Jason D. McEwen, Benjamin Joachimi, Michael P. Hobson
- **Library repository:** https://github.com/alessiospuriomancini/cosmopower
- **License:** GPL-3 with an additional non-commercial research-use condition per the upstream repository
- **Citation:** Spurio Mancini, A., Piras, D., Alsing, J., Joachimi, B., Hobson, M., "CosmoPower: emulating cosmological power spectra for accelerated Bayesian inference from next-generation surveys", MNRAS 511, 1771 (2022), arXiv:2106.03846
- **Redistributed emulator cache files** (this work): `TT_v1.npz`, `TE_v1.npz`, `EE_v1.npz`, deposited under Zenodo record 19741059 (DOI 10.5281/zenodo.19741059) alongside the MTDF source bundle. These `.npz` files are emulator *cache artefacts* generated by training CosmoPower on CLASS spectra; they are redistributed as MTDF reproducibility artefacts only. Users must honour the same non-commercial research-use conditions as the upstream CosmoPower workflow unless a separate licence is explicitly granted.
- **Download:** `scripts/download_cosmopower_models.sh` (SHA-256 verified against the Zenodo deposit).

## Observational Data

All external datasets are downloaded from their official archives via scripts in `scripts/`. They are **not redistributed** in this repository.

### Planck 2018

- **Archive:** ESA Planck Legacy Archive (https://pla.esac.esa.int)
- **Data used:** CMB lensing convergence maps, plik TTTEEE likelihood, distance priors
- **Download:** `scripts/download_planck.sh`
- **Citation:** Planck Collaboration, "Planck 2018 results. VI. Cosmological parameters", A&A 641, A6 (2020), arXiv:1807.06209

### Pantheon+ Type Ia Supernovae

- **Archive:** https://github.com/PantheonPlusSH0ES/DataRelease
- **Data used:** SN Ia distance moduli, covariance matrices, host galaxy properties
- **Download:** `scripts/download_pantheonplus.sh`
- **Citation:** Brout, D. et al., "The Pantheon+ Analysis: Cosmological Constraints", ApJ 938, 110 (2022), arXiv:2202.04077

### DESI VAST Void Catalogues

- **Archive:** DESI data release (https://data.desi.lbl.gov)
- **Data used:** VAST void catalogues (VoidFinder, V2, REVOLVER)
- **Download:** `scripts/download_desi_voids.sh`
- **Citation:** Douglass, K. et al., "DESIVAST: A Catalog of Cosmic Voids", ApJ 978 (2025), arXiv:2409.09419

### ZTF DR2 Cosmology Sample

- **Archive:** https://github.com/ZwickyTransientFacility/ztfcosmo
- **Data used:** Type Ia SN Ia SALT2 light-curve parameters, redshifts, host galaxy masses
- **Download:** `scripts/download_ztf_dr2.sh`
- **Citation:** Rigault, M. et al., "The ZTF DR2 SN Ia Cosmology Sample", arXiv:2409.04346 (2024); Amenouche, M. et al., "ZTF DR2 SN Ia Standardization", arXiv:2409.04344 (2024)

### Foundation DR1 Supernova Survey

- **Archive:** https://github.com/djones1040/Foundation_DR1
- **Data used:** Type Ia SN SALT2 fit parameters and light curves
- **Download:** `scripts/download_foundation_dr1.sh`
- **Citation:** Foley, R. J. et al., "The Foundation Supernova Survey", MNRAS 475, 193 (2018); Jones, D. O. et al., "The Foundation Supernova Survey: Cosmological Constraints", ApJ 881, 19 (2019), arXiv:1811.09286

### KiDS-1000 Weak Lensing

- **Archive:** Leiden Observatory (https://kids.strw.leidenuniv.nl)
- **Data used:** Shear catalogue, photometric redshifts
- **Download:** `scripts/download_kids.sh`
- **Citation:** Kuijken, K. et al., "The fourth data release of the Kilo-Degree Survey", A&A 625, A2 (2019)

### SDSS/BOSS/eBOSS BAO

- **Archive:** SDSS data release (https://www.sdss.org/dr16/)
- **Data used:** BAO distance measurements, growth rate f*sigma8
- **Download:** `scripts/download_bao.sh`
- **Citations:**
  - Alam, S. et al., "Completed SDSS-IV extended Baryon Oscillation Spectroscopic Survey", Phys. Rev. D 103, 083533 (2021), arXiv:2007.08991
  - DESI Collaboration, "DESI 2024 III: Baryon Acoustic Oscillations from Galaxies and Quasars", arXiv:2404.03000

### BOSS DR12 Void Catalogue

- **Archive:** Mao et al. (2017), Vanderbilt Astro
- **Data used:** Void positions and radii for CMB lensing cross-correlation benchmark
- **Download:** `scripts/download_bao.sh`
- **Citation:** Mao, Q. et al., "A cosmic void catalog of SDSS DR12 BOSS galaxies", ApJ 835, 161 (2017), arXiv:1602.02771

### Cosmic Chronometers

- **Archive:** Moresco et al. compilation
- **Data used:** H(z) measurements from differential galaxy ages
- **Download:** `scripts/download_bao.sh`
- **Citation:** Moresco, M. et al., "A 6% measurement of the Hubble parameter at z ~ 0.45", JCAP 05 (2016) 014, arXiv:1601.01701

### Pittordis+2023 Wide Binaries

- **Archive:** Zenodo (https://zenodo.org/records/7932764)
- **Data used:** Wide binary catalogue for gravity sector validation
- **Download:** `scripts/download_pittordis.sh`
- **Citation:** Pittordis, C. and Sutherland, W., "Testing modified-gravity theories via wide binaries", MNRAS 527, 4573 (2024), arXiv:2305.07653
