2. ANALYSIS PIPELINE (in execution order)
================================================================================

STEP 1: Main GLS Analysis
  Script: code/analysis/sn_void_GLS_analysis.py
  Purpose: Core Generalized Least Squares analysis with full covariance

  Method:
    - Cross-match SNe with void catalogs
    - Compute signed distance: d_signed = (r_SN - R_void) / R_void
    - Fit model: μ = μ_ΛCDM + α + γ_env × d_signed + γ_M × step(M*)
    - Test: Δχ² between nested models (1 d.o.f.)

  Output: GLS_analysis_summary.txt (full results documentation)

STEP 2: Pre-registered NGC/SGC Analysis + Selection Function Checks
  Script: code/analysis/sn_void_preregistered_analysis.py
  Purpose:
    - Treat NGC and SGC as separate pre-registered analyses
    - Check selection function (z-distribution, host-mass distribution)
    - Test alternate environment metric (local void density)

  Key findings:
    - NGC has lower mean z (~0.04) vs SGC (~0.08-0.10)
    - NGC shows significant signal, SGC shows null
    - Selection function difference explains asymmetry

STEP 3: Z-Modulation Analysis
  Script: code/analysis/sn_void_z_modulation_analysis.py
  Purpose:
    - Z-matched comparison using Inverse Probability Weighting (IPW)
    - Continuous z-modulation: γ_env = γ₀ + γ₁ × (z - z_mean)
    - Piecewise model: separate γ_env for z < 0.05 and z ≥ 0.05
    - Z-binned analysis

  Key findings:
    - Z-matched SGC still shows null signal → footprint-specific
    - γ₁ is NEGATIVE (weakening with z, as MTDF predicts)
    - Low-z bin (z < 0.04) shows γ = +0.013 (p = 0.003)

STEP 4: Survey Fixed-Effects Analysis
  Script: code/analysis/sn_void_survey_control_analysis.py
  Purpose:
    - Show survey composition in NGC vs SGC
    - Add per-survey fixed effects (15 survey dummies)
    - Test if γ_env survives survey fixed effects
    - Within-survey analysis for largest surveys
    - Leave-One-Survey-Out (LOSO) robustness check

  Key findings:
    - γ_env barely changes with survey FE (Δγ = -0.0004)
    - Low-z remains significant (p = 0.010) with survey FE
    - 12/12 LOSO runs maintain positive γ_env
    - Signal is NOT driven by any single survey

STEP 5: Summary Figure Generation
  Script: code/analysis/sn_void_summary_figure.py
  Purpose: Create publication-ready figure with key results

  Output:
    - sn_void_summary_figure.png (300 dpi raster)
    - sn_void_summary_figure.pdf (vector format)

6. INTERPRETATION & CAVEATS
================================================================================

INTERPRETATION:
  - Positive γ_env means SNe OUTSIDE voids appear FAINTER
  - Equivalently: SNe in/near voids appear ~0.01 mag BRIGHTER
  - Effect is concentrated at low z (z < 0.05)
  - Consistent with MTDF stress coherence scale prediction
  - Independent of known systematics (host-mass, survey)

CAVEATS (stated explicitly):
  1. Only 4 SNe strictly inside void boundaries in low-z sample
     → Inference is from continuous signed-distance metric

  2. NGC/SGC asymmetry: strong signal in NGC, null in SGC
     → Explained by z-distribution difference (NGC has lower mean z)
     → Could also reflect better void coverage in NGC

  3. VoidFinder shows weaker signal than REVOLVER/VIDE
     → Different void finding algorithms have different sensitivity

  4. Combined sample result (p ~ 0.04-0.09) is suggestive
     → NGC footprint (p < 0.01) is more significant but is subgroup

8. MERGED-SAMPLE HARDENING SUITE (April 2026)
================================================================================

STEP 6: Build Merged Sample
  Script: code/analysis/sn_void_hardening/build_merged_sample.py
  Purpose: Merge Pantheon+ (1533) + ZTF DR2 (2650) + Foundation DR1 (5 unique)
  Method: 5" sky-position cross-match, Pantheon+ priority, Tripp standardisation
          for ZTF (alpha=0.148, beta=3.112), diagonal cov with sigma_int=0.12
  Output: merged_sn_ia.csv (4188 SNe), merged_cov_diag.npy

STEP 7: Six Hardening Tests
  Orchestrator: code/analysis/sn_void_hardening/run_merged_hardening.py
  Shared code:  code/analysis/sn_void_hardening/common.py

  Test 1 (scrambled voids):     test_scrambled_voids.py
    Result: constant-gamma null (Dchi2 < 1.4, all catalogues)

  Test 2 (z-transition):        test_fake_z_transition.py
    Result: z=0.04 piecewise Dchi2=36-39, p<0.001, all catalogues (HEADLINE)

  Test 3 (population controls): test_population_controls.py
    Result: gamma shifts < 0.3 sigma with x1/c covariates

  Test 4 (wrong-sign metric):   test_wrong_sign_metric.py
    Result: symmetric (consistent with null constant model)

  Test 5 (alt metrics):         test_alt_environment_metrics.py
    Result: IDW metric significant (p<0.013, all catalogues)

  Test 6 (cross-catalogue):     test_cross_catalogue_overlap.py
    Result: top-30 pairwise overlap 60-77%, same physical SNe drive signal
