1. ANALYSIS PROTOCOL
================================================================================

Data:
  - Pantheon+SH0ES: 1701 Type Ia SNe with 1701×1701 STAT+SYS covariance
  - DESIVAST DR1: VoidFinder, V2/REVOLVER, V2/VIDE void catalogs
  - Coordinate system: Comoving Mpc/h (Ω_m = 0.315, h = 1)

Cuts:
  - Redshift: 0.02 < z ≤ 0.157 (PV cut + DESIVAST limit)
  - Sample after cuts: 564 SNe

Model:
  μ_i = μ_ΛCDM(z_i) + α + γ_env × d_signed + γ_M × step(log M* ≥ 10)

where d_signed = (r_SN - R_void) / R_void is the signed distance to the
nearest void boundary. Negative values indicate SNe inside voids.

Test statistic: Δχ² between nested models (with/without γ_env term)

IMPORTANT NOTE: Only 4 SNe strictly inside void boundaries in this low-z
sample. Binary inside/outside tests have low power; inference is mainly
driven by the continuous signed-distance metric.

  All three void finders show CONSISTENT direction:

significant (p = 0.039), while VoidFinder and VIDE are marginal.

4. KEY ROBUSTNESS FACTS
================================================================================

FACT 1: Consistent sign across all three void finders

Same sign and similar magnitude (0.003 to 0.005 mag per unit signed distance)
across ALL THREE independent void catalogs.

A random fluctuation is unlikely to produce consistent sign across three
independent void finding algorithms with different methodology:
  - VoidFinder: maximal sphere fitting
  - REVOLVER: watershed transform
  - VIDE: ZOBOV/Voronoi tessellation

FACT 2: Highly significant signal in NGC footprint (key result)

When using proper footprint-based region assignment:

│ Catalog     │ NGC γ_env            │ NGC p-value          │
├─────────────┼──────────────────────┼──────────────────────┤

│ VIDE        │ +0.0111 ± 0.0040     │ 0.0058 ★★            │
│ VoidFinder  │ +0.0044 ± 0.0040     │ 0.277                │
└─────────────┴──────────────────────┴──────────────────────┘

★★ = p < 0.01 (highly significant)

The NGC footprint shows γ_env ~ +0.01 mag per unit signed distance for
both REVOLVER and VIDE, with p < 0.01. This is 2-3× stronger than the
combined sample result and survives at the 1% level.

Note: SGC footprint shows null result for all three catalogs.
This NGC/SGC asymmetry is a concern that warrants investigation:
  - Could indicate better void coverage in NGC
  - Could indicate NGC-specific large-scale structure
  - Could be a statistical fluctuation (less likely given 2 independent catalogs)
