# Pittordis+2023 Wide Binary Data

> **Author:** Ingo Mesche | **Affiliation:** Independent Researcher, Malta | **Framework:** MTDF V74


This directory should contain the wide binary catalogue from Pittordis & Sutherland (2023).

## Reference

Pittordis, C. & Sutherland, W. (2023). "Testing modified-gravity theories via wide binaries and GAIA eDR3."
Monthly Notices of the Royal Astronomical Society, 527(3), 4573-4601.
DOI: 10.1093/mnras/stad3474

## Download

From the repository root:

```bash
bash scripts/download_pittordis.sh
```

This downloads the catalogue (~164 MB) from CDS/VizieR and places the files in this directory.

## Expected Files

After download, this directory should contain the wide binary catalogue FITS/CSV files used by `gravity/code/step19_wide_binaries.py`.
